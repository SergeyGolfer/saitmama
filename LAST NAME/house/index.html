Эскизно-планировочный этап
  • обмерный чертеж с привязкой коммуникаций;
  • несколько вариантов планировочных решений;
  • план расстановки мебели;
  • план монтажа/демонтажа перегородок;
  • план монтажа инженерных коммуникаций;
  • окончательный вариант планировки с указанием точных размеров.
Рабочий этап
  • план расположения санитарно-технического оборудования с указанием размеров и сантехнических привязок;
  • план потолка с указанием используемых материалов для каждого отдельного узла;
  • план монтажа осветительной техники;
  • привязка выпусков освещения;
  • план выключателей для каждой группы светильников;
  • план размещения силовых и слаботочных розеток;
  • план размещения элементов электрического подогрева пола (опционально);
  • план заполнения дверных проемов;
  • план размещения климатического оборудования;
  • развертки всех помещений с указанием отделочных материалов, расположения мебели, оборудования;
  • ведомости по отделочным материалам;
  • эскизы на мебель для индивидуального изготовления;
  • эскизы на декоративные элементы, конструкции, перегородки витражи;
  • спецификации на чистовые отделочные материалы, осветительное оборудование и др.
Визуализация
  • эскизов стилевого и колористического оформления помещений;
  • по несколько ракурсов визуализации на каждую комнату;
  • расположение мебели, светильников и оборудования;
  • прорисовки помещений санузлов с плиткой и сантехническим оборудованием;
